# DailyRewards

## Overview
DailyRewards is a plugin I made that allows servers to automatically grant rewards to players who log in consecutively.  
The plugin has a fully functional default setup, but can be fully customized through the config.yml file.  

The default max number of consecutive logins a player can have before running out of rewards is 30, however this is capped at a maximum of 54.  
Once a player hits the max consecutive logins, they are granted the highest reward for each following consecutive login.  
If a player misses a day they are reset to day 1.  

## Commands
/daily. This opens the daily menu, and can be used by all players.  
/reload. This reloads the config.yml file, and can be used by players with permission.  

## Permissions
"daily_rewards.reload". Grants permission to reload the config.yml file
